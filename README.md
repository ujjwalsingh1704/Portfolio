## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/ujjwalsingh1704/Portfolio.git
```

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm start
```

### Build for Production

```bash
npm run build
```
